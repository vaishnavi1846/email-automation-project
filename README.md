# Email Automation Flask Project

A Flask-based web application that sends bulk emails using CSV data and supports attachments.

## Features
- Bulk email sending
- CSV-based email management
- Attachment support
- SMTP integration
- Environment variable security

## Tech Stack
- Python
- Flask
- HTML
- SMTP
- Git

## How to Run

1. Create virtual environment
2. Install requirements:
   pip install -r requirements.txt
3. Add .env file
4. Run:
   python app.py
